#Composer WP Environment.

WordPress Environment for PHP Built-in Server.

## Getting Started

### 1. Install MySQL, jq and composer.

```
$ brew install mysql jq composer
```

#### 2. Create Project

```
$ composer create-project torounit/composer-wp path/to/project
```

### 3. Start Development!

```
$ cd path/to/project
$ bin/bootstrap.sh
```
